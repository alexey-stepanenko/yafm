//

#ВставитьЭлементОписание
// ЭЛЕМЕНТ	<Имя элемента>	<Тип элемента>
//   <Имя элемента> - Имя элемента;
//   <Тип элемента> - Тип элемента:
//     * ДЕКОРАЦИЯФОРМЫ
//     * ГРУППАФОРМЫ
//     * КНОПКАФОРМЫ
//     * ТАБЛИЦАФОРМЫ
//     * ПОЛЕФОРМЫ
//
// РОДИТЕЛЬЭЛЕМЕНТА	<Имя элемента>	<Родитель>
//   <Имя элемента> - Имя элемента, указанное в блоке ЭЛЕМЕНТ;
//   <Родитель>     - Имя родительского элемента;
//
// ПОСЛЕДУЮЩИЙЭЛЕМЕНТ	<Имя элемента>	<Элемент>
//   <Имя элемента> - Имя элемента, указанное в блоке ЭЛЕМЕНТ;
//   <Элемент>      - Имя элемента, перед которым должен быть вставлен новый элемент;
//
// СВОЙСТВОЭЛЕМЕНТА	<Имя элемента>	<Имя свойства>	<Значение свойства>
//   <Имя элемента>      - Имя элемента, указанное в блоке ЭЛЕМЕНТ;
//   <Имя свойства>      - Имя свойства. См. дополнительно синтакс-помощник по
//     свойствам элемента формы;
//   <Значение свойства> - Значение свойства. См. дополнительно синтакс-помощник по
//     свойствам элемента формы. Для свойств с типами "Цвет", "СочетаниеКлавиш", "Рамка"
//     указывается внутреннее имя, само определение значения свойства идет в
//     соответствующем блоке: ЦВЕТ, СОЧЕТАНИЕКЛАВИШ, РАМКА;
//
// СОБЫТИЕЭЛЕМЕНТА	<Имя элемента>	<Имя события>	<Имя процедуры-обработчика>
//   <Имя элемента>              - Имя элемента, указанное в блоке ЭЛЕМЕНТ;
//   <Имя события>               - Имя события. См. дополнительно синтакс-помощник по
//     событиям элемента формы;
//   <Имя процедуры-обработчика> - Имя процедуры обработчика без параметров, находится
//     в модуле формы;
//
// Пример:
// ЭЛЕМЕНТ	Группа1	ГРУППАФОРМЫ
// СВОЙСТВОЭЛЕМЕНТА	Группа1	Вид	ОбычнаяГруппа
// СВОЙСТВОЭЛЕМЕНТА	Группа1	Отображение	ОбычноеВыделение
// СВОЙСТВОЭЛЕМЕНТА	Группа1	ОтображатьЗаголовок	Истина
// СВОЙСТВОЭЛЕМЕНТА	Группа1	Группировка	ГоризонтальнаяВсегда
// СВОЙСТВОЭЛЕМЕНТА	Группа1	Заголовок	Заголовок группы 1
//
// ЭЛЕМЕНТ	Реквизит1	ПОЛЕФОРМЫ
// РОДИТЕЛЬЭЛЕМЕНТА	Реквизит1	Группа1
// СВОЙСТВОЭЛЕМЕНТА	Реквизит1	ПутьКДанным	Реквизит1
// СВОЙСТВОЭЛЕМЕНТА	Реквизит1	Вид	ПолеВвода
// СОБЫТИЕЭЛЕМЕНТА	Реквизит1	ПриИзменении	ОбработчикПриИзменении
//
ЭЛЕМЕНТ	<Имя элемента>	<Тип элемента>
РОДИТЕЛЬЭЛЕМЕНТА	<Имя элемента>	<Родитель>
ПОСЛЕДУЮЩИЙЭЛЕМЕНТ	<Имя элемента>	<Элемент>
СВОЙСТВОЭЛЕМЕНТА	<Имя элемента>	<Имя свойства>	<Значение свойства>
СОБЫТИЕЭЛЕМЕНТА	<Имя элемента>	<Имя события>	<Имя процедуры-обработчика>

#ВставитьЦветОписание

//   <Имя цвета>      - Внутренее имя цвета. Используется для указания значения в свойствах элемента
//     с типом "Цвет";
//   <Описание цвета> - Описание цвета. Используются несколько нотаций:

// Пример:

// ЦВЕТ	Красный	255,0,0

//
